# Reactivue

Reactivue is a modern frontend framework that provides a simple and efficient way to build user interfaces. It is designed to be highly scalable and flexible, making it suitable for a wide range of applications.

## Features
* Reactive components
* Virtual DOM
* Efficient rendering
* Support for TypeScript
* Compatible with Vite and Tailwind

## Getting Started
1. Install the dependencies: `npm install`
2. Copy the environment variables: `cp .env.example .env`
3. Start the development server: `npm start`
